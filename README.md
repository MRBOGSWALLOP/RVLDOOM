# RVLDOOM - Drag and Drop Doom for Wii with native controller support
 This is a well packaged version of the excellent PrBoom port for Libretro. I wanted to play DOOM on my Wii using simple methods. 
 WiiDoom on the homebrew wiki has no Gamecube controller port and the chocolate doom port is just too much work. This package includes:
 - Freeware wad of DOOM 1
 - A comfortable binding for the gamecube controller 
 - Mod support for .IWAD and .PWAD
 - Resolution set for a crt in standard resolution (composite)
 - bloat reduction of retroarch cores
 - prboom.wad in roms

I woud like to these QoL improvements:
 - .DOL for each game to launch directly from homebrew channel
 - Forwarder channels to launch from system menu or usbloaderGX
 - Improve SIGIL stability
